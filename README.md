# contidio-widget
Include a single or multiple Contidio Assets or Overviews into your existing website

### Development

Prerequisites:
```
npm install
```

To build the widget:
```
npm build
```

To watch for changes:
```
npm run watch
```

### Example

Start up the webserver:
```
npm run example
```

And go to http://127.0.0.1:8080/ to see a list of all examples
